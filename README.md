# strawberry-robot
Code for a small two-wheeled robot

Feather Outputs
* 2 motors - total 4 GPIO, 2 PWD
* 1 servo - 1 PWD
* 1 OLED screen - Vin, GND, SCL, SDA (2 GPIO)

Feather Inputs (via BlueFruit Connect -> Controller -> Control Pad)
* Drive inputs - buttons 5 -8
